* stumpwm-config
My StumpWM configuration.

** Installation
#+BEGIN_SRC shell
git clone https://github.com/manzaltu/stumpwm-config ~/.stumpwm.d
cd ~/.stumpwm.d
git submodule update --init
#+END_SRC

** Getting updates
You can update the configuration by running the following command inside the cloned dir:
#+BEGIN_SRC shell
git pull -r
#+END_SRC
